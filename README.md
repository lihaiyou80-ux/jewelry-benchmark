# 标品视频号 · 基建对标诊断

面向广告投放团队的头部客户基建对标诊断站点。用户输入视频号名称，站点会自动匹配该视频号所属品类，并对比同品类头部客户（TOP1/TOP2/TOP3 或自选对标）的基建差异。

## 访问

⚠️ 站点受访问口令保护。请联系管理员获取访问口令。

## 功能

- 🔒 **AES-GCM 加密**：数据在客户端解密，无口令即无法查看
- 🎯 **匿名对标**：默认对比品类 TOP1/TOP2/TOP3（不露出头客名字）；也支持输入其他视频号名做自定义对标
- 📊 **四大板块**：
  1. 一句话总结 + 核心归因（基于 f23 时间序列前 7 vs 近 7 自动生成）
  2. 关键差距诊断（vs 每个对标）
  3. 基础基建 / 产品使用 / 投放目标 / 内容力 · 同尺寸并排对比
  4. 消耗趋势诊断（10 图 + 前后期指标对比表）

## 覆盖范围

- **371 个标品视频号**，共 15 个品类
- 数据窗口：14 日（详见页面顶部）

## 技术

- 纯静态站点（HTML + CSS + 原生 JS，无框架依赖）
- 数据加密：AES-256-GCM + PBKDF2-SHA256（250000 iter）
- 部署：GitHub Pages
